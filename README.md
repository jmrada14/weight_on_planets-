Weight on planets is a simple CLI built with [Rust](https://www.rust-lang.org/). 
to calculate the weight of a person on different planets and moons.

## Planets and moons supported:
- earth 
- jupiter 
- mars
- mercury 
- neptune 
- saturn 
- uranus
- venus 

## Tools:
- [Rust](https://www.rust-lang.org/)
